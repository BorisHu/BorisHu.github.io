Overview

This is the prototype version of the final project. Although it is the prototype version, I think it have all the features I am expecting for the final version.

Development Process

  1. I have first build up the parts that I am not familar with such as firebase and use lit-html to render pages.
  2. Then I modified an existing bullethell game to suit this project.
  3. Finally, I combined different parts of the code and made this prototype.

Issue Deep-Dive

  • The biggest problem I have encountered is the anon login and display ID at the same time.
  • To solve this, I changed the display name of the anon user in firebase when the user input their name.

Ideas and Future
