Final Project Proposal!

Project Concept:

I would like to combine this project with the HCDE 438 project to make a bullet hell web-game that the character could be controlled by the joystick and buttons. The HP of the character will be displayed on the seven segment display, the score of the player will be showed on the 4 digit seven-segment display.

Anticipated Bill of Materials:

  • Arduino Uno
  • Joystick (for control character move)
  • Buttons (for control character shoot)
  • 7 segment display (for display health and score)
  • Resistors
  • Wires
  • Breadboard
  • A bullet hell game made by P5.js (for game)

Timeline and contingency plans:

  • By March 2nd: Try using each electronic parts for tasks to prepare for the project.
  • By March 3rd: Finish coding part of the bullet hell game made by P5.js and connect to arduino.
  • By March 7th: Build the Physical Prototype and test each electronic part work with the game. Debug in class.
  • By March 9th: Final Presentation, webpage documentation.

Plan B:

If something goes wrong with using physical parts to control or display information in the game, I will choose to use the computer keys or display.
